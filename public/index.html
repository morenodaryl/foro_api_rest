API REST

Api desarrollada en ruby on rails para el foro de desarrollo movil UNI RUACS.

Recursos de contactos

URI HTTP Acción
/contactos GET Ver todos los contactos.
/contactos POST Agregar un nuevo contacto.
Para crear un nuevo recurso tendremos que mandar los siguientes parametros:
{
"contacto": {
"nombre": "Nombre del contacto",
"correo": "correo de la persona",
"edad": "edad de tipo integer"
}
}
/contactos/:id GET Ver un contacto con el id especificado en la URI
/contactos/:id PUT | PATCH Actualizar un recurso. para actualizar un contacto se debe de seguir la misma lógica del metodo post.
/contactos/:id DELETE Eliminar recurso. para eliminar un contacto se hace solo especificando el id por la URI.
